Responsive Classes

The Bootstrap 3 grid system has four classes:

  • .col-xs-* (extra small devices - screen width less than 768px)
  • .col-sm-* (small devices - screen width equal to or greater than 768px)
  • .col-md-* (medium devices - screen width equal to or greater than 992px)
  • .col-lg-* (large devices - screen width equal to or greater than 1200px)

Responsive Columns (xs)

Proportioned for extra small devices and all larger screen widths.
