Bootstrap 3 Alerts

Bootstrap provides an easy way to create predefined alert messages.

Tip: Plugins can be included individually (using Bootstrap's individual "alert.js" file), or all at once (using "bootstrap.js" or "bootstrap.min.js").

Alerts Close Button

The <a> element with class="close" and data-dismiss="alert" is used to close the alert box.

The .alert-dismissible class adds some extra padding to the close button.

(To help improve accessibility for people using screen readers, you should include the aria-label="close" attribute, when creating a close button.)
