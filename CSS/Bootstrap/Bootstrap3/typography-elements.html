Bootstrap 3 Typography - Elements

Mark <mark>

Use the <mark> element to highlight text.

Abbreviation <abbr>

The WHO was founded in 1948

Delete <del>

Use the <del> element to indicate deleted text.

Strike-through <s>

Use the <s> element to indicate text that is no longer relevant.

Inserted <ins>

Use the <ins> element to indicate inserted text.

Underline <u>

Use the <u> element to indicate underlined text.

Strong <strong>

Use the <strong> element to indicate bold text.

Emphasis <em>

Use the <em> element to indicate italic text.

Address <address>

Use the <address> element to present contact information:

Jon Doe
Example.com
Box 564
Disneyland
USA

Blockquote <blockquote>

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.

From WWF's website

To show the quote on the right use the class .blockquote-reverse:

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.

From WWF's website

Unordered List <ul>

Use the <ul> element to indicate an unordered list:

  • Coffee
  • Tea
    • Black tea
    • Green tea
  • Milk

Ordered List <ol>

Use the <ol> element to indicate an ordered list:

  1. Coffee
  2. Tea
  3. Milk

Description List <dl>

The <dl>element indicates a description list:

Blockchain
A chain of blocks, each block containing digitally-signed and verified data grouped by timestamp.
Permissioned blockchain
A private/closed blockchain where an access control layer governs who has access to the network.

Code <code>

Inline snippets of code should be embedded in the code element:

The following HTML elements: span, section, and div defines a section in a document.

Multiple Code Lines <pre>

For multiple lines of code, use the pre element:

Text in a pre element is displayed in a fixed-width font, and it preserves both      spaces and
				line breaks.

Sample Output <samp>

To indicate sample output from a computer program, use the <samp> element:

This text is output from a computer program....

Math <var>

To indicate variables, use the <var> element:

x = ab + y

Keyboard Inputs <kbd>

To indicate input that is typically entered via the keyboard, use the kbd element:

Use ctrl + p to open the Print dialog box.
