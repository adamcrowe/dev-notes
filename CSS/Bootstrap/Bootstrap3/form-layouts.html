Bootstrap 3 Form Layouts

Three Layouts

Bootstrap provides three types of form layouts:

  • Vertical form (default)
  • Horizontal form
  • Inline form

Standard rules for all three form layouts:

  • Wrap labels and form controls in <div class="form-group"> (needed for optimum spacing)
  • Use the .form-control class with all textual <input>, <textarea>, and <select> elements

Inline Form

Use the .form-inline class with <form> elements

Note: This only applies to forms within viewports that are at least 768px wide

Horizontal Form

On small screens (767px and below), it will transform to a vertical form (labels are placed on top of each input).

Additional rules for a horizontal form:

  • Use the .form-horizontal class with the <form> element
  • Use the .control-label class with all <label> elements

Tip: Use Bootstrap's predefined grid classes to align labels and groups of form controls in a horizontal layout.
