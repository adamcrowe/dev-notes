Bootstrap 4 Dropdowns

A dropdown menu is a toggleable menu that allows the user to choose one value from a predefined list

Basic Dropdown

Use a button or a link with a class of .dropdown-toggle and the data-toggle="dropdown" attribute. The .caret class creates a caret arrow icon (), which indicates that the button is a dropdown. Add the .dropdown-menu class to a <ul> element to actually build the dropdown menu. Add the .dropdown-item class to each menu item.
