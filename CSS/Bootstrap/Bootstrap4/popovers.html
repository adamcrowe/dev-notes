Bootstrap 4 Popovers

Popovers are similar to tooltips; it is a pop-up box that appears when the user clicks on an element

Overview

Popovers rely on the 3rd party library Popper.js for positioning. You must include popper.min.js before bootstrap.js or use bootstrap.bundle.min.js / bootstrap.bundle.js which contains Popper.js in order for popovers to work!

Note: Popovers are opt-in for performance reasons, so you must initialize them yourself. must be initialized with jQuery: select the specified element and call the popover() method.

The following code will enable all popovers in the document:

<script>
$(document).ready(function(){
  $('[data-toggle="popover"]').popover();
});
</script>

The following code will enable a popover on the specified element:

<script>
$(document).ready(function(){
  $('#myPopover').popover();
});
</script>

The following code will enable a popover on the specified selector:

<script>
$(document).ready(function(){
  $('body').popover({title: "Hooray!", selector: ".btn"});
});
</script>

Basic Popover

Add the data-toggle="popover" attribute to an element. Use the title attribute to specify the header text of the popover, and use the data-content attribute to specify the text that should be displayed inside the popover's body:

Title

The title option (default: "") specifies the text that should be displayed inside the popover.

Trigger

The trigger option (default: "hover focus") specifies how the popover is triggered.

Possible values (To pass multiple values, separate them with a space):

  • "click" - Trigger the popover with a click
  • "hover" - Trigger the popover on hover
  • "focus" - Trigger the popover when it gets focus (by tabbing or clicking .e.g

Positioning

The placement option (default: "top") specifies the popover position.

Tip: You can also use the data-placement attribute with a value of "auto", which will let the browser decide the position of the popover. For example, if the value is "auto left", the popover will display on the left side when possible, otherwise on the right.

Disabled

Animation

The animation option (default: true) specifies whether to add a CSS fade transition effect to the popover.

Container

The container option (default: false) appends the popover to a specific element.

Delay

The delay option (default: 0) specifies the number of milliseconds it will take to show and hide the popover.

This popover will take 1000 milliseconds to show when you click it, and 1000 milliseconds to hide on click leave.



This popover will take 500 milliseconds to show when you click it, and 100 ms to hide on click leave.

HTML

The html option (default: false) specifies whether to accept HTML tags in the popover.

When set to false (default), jQuery's text() method will be used. Use this if you are worried about XSS attacks.

Template

Base HTML to use when creating the popover. The popover's title will be inserted into the element having the class .popover-inner and the element with the class .popover-arrow will become the popover's arrow. The outermost wrapper element should have the .popover class.

<div class="popover" role="popover">
  <div class="popover-arrow"></div>
  <div class="popover-inner"></div>
</div>

Viewport

Keeps the popover within the bounds of this element (default: {selector: "body", padding: 0}).

Example: viewport: '#viewport' or {selector: '#viewport', padding: 0}

If a function is given, it is called with the triggering element DOM node as its only argument. The this context is set to the popover instance.

Popover Options and Methods

See: https://getbootstrap.com/docs/4.0/components/popovers/

Popover will show here

  • .popover('show') Shows the popover
  • .popover('hide') Hides the popover
  • .popover('toggle') Toggles the popover
  • .popover('destroy') Hides and destroys the popover

Popover Events

Popover will show here

The show.bs.popover event occurs when the popover is about to be shown.

The shown.bs.popover event occurs when the popover is fully shown.

The hide.bs.popover event occurs when the popover is about to be hidden.

The hidden.bs.popover event occurs when the popover is fully hidden.
