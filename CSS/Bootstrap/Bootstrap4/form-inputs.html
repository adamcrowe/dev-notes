Disabled

Add the disabled boolean attribute on an input to prevent user interactions and make it appear lighter.

Add the disabled attribute to a <fieldset> to disable all the controls within.

Read-only

Add the readonly boolean attribute on an input to prevent modification of the input’s value. Read-only inputs appear lighter (just like disabled inputs), but retain the standard cursor.

Read-only (Plain Text)

If you want to have <input readonly> elements in your form styled as plain text, use the .form-control-plaintext class to remove the default form field styling and preserve the correct margin and padding.

Layout

Bootstrap applies display: block and width: 100% to almost all our form controls, forms will by default stack vertically. Additional classes can be used to vary this layout on a per-form basis.

Form Groups

The .form-group class adds structure to forms: it encourages the proper grouping of labels, controls, optional help text, and form validation messaging.

Form Grid

Form Row (Compact)

Swap .row for .form-row, a variation of the standard grid row that overrides the default column gutters for tighter and more compact layouts.

Horizontal Form

Use the .row class to form groups and the .col-*-* classes to specify the width of your labels and controls. Add the .col-form-label to each <label> so they're vertically centered with their associated form controls.

Help Text

Help text below inputs can be styled with .form-text. This class includes display: block and adds some top margin for easy spacing from the inputs above.

Help text should be explicitly associated with the form control it relates to using the aria-describedby attribute. This will ensure that screen readers will announce this help text when the user focuses or enters the control.
