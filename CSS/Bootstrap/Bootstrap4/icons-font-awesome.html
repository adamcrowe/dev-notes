Bootstrap 4 Icons: Font Awesome

See: https://fontawesome.com/cheatsheet

Font Awesome Syntax

An icon is inserted with the following syntax:

<span class="fa fa-star"> (default)
<span class="far fa-star"> (regular)
<span class="fas fa-star"> (solid)
