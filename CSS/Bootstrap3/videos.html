Responsive Embed

Classes can be applied directly to <iframe>, <embed>, <video>, and <object> elements.

Create a responsive video by adding an .embed-responsive-item class to an <iframe> tag (the video will then scale nicely to the parent element). The containing <div> can also define the aspect ratio of the video, either 16:9 or 4:3:

16:9 Aspect Ratio .embed-responsive-16by9

4:3 Aspect Ratio .embed-responsive-4by3
