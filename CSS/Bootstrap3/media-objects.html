Bootstrap Media Objects

Bootstrap provides an easy way to align media objects (like images or videos) to the left or to the right of some content.

Basic Media Object

Use the .media-left or .media-right class to align a media object. Text that should appear next to the image is placed inside a container with .media-body class.
